Copion HTML

HTML est un langage des balises. Il permet de faire des sites web, qu'on ouvre dans un navigateur comme chrome ou firefox.
Le HTML est le squelette de la page, alors que le CSS est l'habillage, et le JavaScript est le dynamisme.

En HTML, chaque élément de la page web visible est codé grâce à des balises, qui délimitent le début et la fin de l'élement: 

          


Hello world!

Chacune de ces balises est fortement sémantique, elle permet d'explique le rôle d'un élément. 

        


Hello world!



Aujourd'hui il fait beau



mentions légales

On peut voir la page HTML comme un grand arbre de balises imbriquées les unes dans les autres. Ou comme un jeu de légo, où plusieurs pièces identiques et/ou différentes s'encastrent les unes dans les autres.
C'est une composition de balise, très déclarative.

  • div, p, h1, h6...
  • ul, form, table...
Les principales balises "inline":
  • strong, em...
  • span, img...
  • input, select...
