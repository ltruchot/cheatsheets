Copion HTML

HTML est un langage de balises.... bla bla bla blab.... que permet de faire des sites web

Pour coder en HTML on utilise une série de balises qui délimitent le code... bla bla bla

Il existe deux grands type de balises: "block" et "inline"

Les principales balises "block":
  • main, header, footer...
  • div, p, h1, h6...
  • ul, form, table...
Les principales balises "inline":
  • strong, em...
  • span, img...
  • input, select...
